New transition rendering — Reel applies transitions via pure TransitionTransformFn instead of the previous string-based system, enabling fully custom slide animations. Built-in transitions (slide, fade, flip, cube, zoom) available as tree-shakeable imports. SoundProvider and useSoundState for shared mute/unmute context. enableGestures prop to disable touch/mouse drag on Reel. Re-exports core utilities: content loading, preloading, sound controller, LRU cache. useFullscreen hook — bridges core fullscreenSignal to React with request/exit/toggle helpers. BREAKING: Reel transition prop no longer accepts string names — pass a TransitionTransformFn directly (e.g. slideTransition instead of 'slide').
