# Changelog

Release history for all `@reelkit/*` packages. Site render workspace [CHANGELOG.md](https://github.com/KonstantinKai/one-item-slider-lib/blob/main/CHANGELOG.md) at runtime; source of truth in repo, generated by `nx release` from conventional commits.

## How it's generated

- Independent versioning per package via `nx release` + conventional commits.
- Tags use independent format: `@reelkit/core@1.0.0`.
- Version bumps from conventional commits (`feat:` → minor, `fix:` → patch).
- GitHub Actions auto-publish to npm on tag push.

## Convention

Each package version block in `CHANGELOG.md` follow:

```markdown
## @reelkit/<package>@<version> (YYYY-MM-DD)

### 🚀 Features

- New feature line

### 🩹 Fixes

- Bug fix line

### 🧱 Updated Dependencies

- Updated <peer> to <version>
```
